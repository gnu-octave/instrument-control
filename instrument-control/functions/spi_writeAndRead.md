<dt class="deftypefn" id="index-spi_005fwriteAndRead"><span class="category-def">Loadable Function: </span><span><code class="def-type"><var class="var">rddata</var> =</code> <strong class="def-name">spi_writeAndRead</strong> <code class="def-code-arguments">(<var class="var">spi</var>, <var class="var">wrdata</var>)</code><a class="copiable-link" href="#index-spi_005fwriteAndRead"></a></span></dt>
<dd> 
<p>Write data to a spi slave device and then read same number of values.
</p> 
<h4 class="subsubheading" id="Inputs"><span>Inputs<a class="copiable-link" href="#Inputs"></a></span></h4>
<p><var class="var">spi</var> - instance of <var class="var">octave_spi</var> class.<br><var class="var">wrdata</var> - data, of type uint8, to be written to the slave device.<br>
</p> 
<h4 class="subsubheading" id="Outputs"><span>Outputs<a class="copiable-link" href="#Outputs"></a></span></h4>
<p>Upon successful completion, spi_writeAndRead() shall return the bytes read.
